◆ addToCalendar()

addToCalendar ( )

Generates and forces the download of an ICS calendar event for a movie release.

This method constructs a file in iCalendar (.ics) format to allow the user to add a movie's release date to their personal calendar.

  • The process follows these steps:
  1. Clears the output buffer to prevent file corruption.
  2. Retrieves the movie ID and user context.
  3. Fetches movie details from the database with a redirect on failure.
  4. Configures event timing (defaults to 20:00, 2-hour duration).
  5. Sanitizes and formats text data according to the iCalendar standard.
  6. Constructs the VEVENT structure with a unique UID.
  7. Sends HTTP headers to force file download.
  • Author
    Etienne
    Parameters
    int$idMovie identifier (retrieved via request context).
    Returns
    void Streams the file directly to the browser and terminates the script.
